# VUE-CLI 설정

## vue/cli setup
```
sudo npm install -g @vue/cli // VUE-CLI 처음사용 하시는 분들이라면 필수적으로 설치해야 한다.
```

## Project setup
```
yarn install
or
npm install
```

### Compiles and hot-reloads for development
```
yarn run serve
or
npm run serve
```

### Compiles and minifies for production
```
yarn run build
or
npm run build
```

### Run your tests
```
yarn run test
or
npm run test
```

### Lints and fixes files
```
yarn run lint
or
npm run lint
```
